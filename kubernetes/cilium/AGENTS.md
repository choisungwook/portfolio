# Cilium CNI hands-on

이 프로젝트는 cilium CNI 학습을 위해 공부한 내용을 정리했습니다.

## 실습환경

실습환경은 MacOS를 사용했으며, kubernetes는 kind cluster를 사용했습니다. talos-config는 무시하세요.

## 디렉터리 구조

```bash
.
├── examples # 실습 예제
│   ├── README.md
│   ├── networkpolicy # networkpolicy 예제
│   └── basic.md # cilium 기초 예제
├── imgs # markdown 파일 작성에 필요한 image파일 모음
└── install # Cilium CNI 실습을 위한 정리파일 모음
```
