# 미래 AI Agent 작업 목록

이 문서는 미래의 AI agent가 이 프로젝트에서 수행할 작업을 정리한다.

## 레이아웃/디자인 확장

추가/수정 절차는 [update-layout.md](./update-layout.md)를 참고한다.

- 디자인 추가 시: `public/designs/{name}.html` 생성 → `index.astro`의 designs 배열에 항목 추가
- 레이아웃 추가 시: `public/layouts/{type}-{variant}.html` 생성 → `index.astro`의 layouts 배열에 항목 추가

## 향후 아이디어

- 애니메이션 프리셋 (fade-in, slide-up)
- 차트/그래프 레이아웃
- 인쇄용 레이아웃 (PDF 최적화)
