Cilium + Hubble 아키텍처

📦
Pod A
요청을 보내는 클라이언트
→
🔍
Cilium Agent
eBPF 기반 패킷 처리 및 관측
→
📊
Hubble
플로우 로그 수집 및 시각화
→
📦
Pod B
요청을 받는 서버
모든 패킷은 Cilium의 eBPF 프로그램을 통과하며, Hubble이 실시간으로 플로우를 기록한다
