CI/CD 파이프라인 구축 단계

1
코드 푸시
개발자가 feature branch에 코드를 push한다
2
빌드 & 테스트
GitHub Actions가 자동으로 빌드와 단위 테스트를 실행한다
3
이미지 빌드
Docker 이미지를 빌드하고 ECR에 푸시한다
4
스테이징 배포
ArgoCD가 스테이징 환경에 자동 배포한다
5
프로덕션
승인 후 프로덕션에 Blue/Green 배포한다
