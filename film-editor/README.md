# Film Editor - 필름 시뮬레이션 사진 에디터

## 요약

- **웹 브라우저에서 동작하는 필름 효과 사진 에디터**입니다. 서버 없이 클라이언트에서 모든 이미지 처리를 수행합니다.
- 20개 필름 프리셋을 제공합니다. Kodak Portra, Fuji Velvia, Ilford HP5 등 실제 필름 특성을 시뮬레이션합니다.
- Before/After 비교, 노출/채도/색온도 등 기본 보정, 그레인/비네팅/페이드 효과를 지원합니다.
- React + TypeScript + Vite + Tailwind CSS 스택이고, Canvas API로 픽셀 단위 이미지 처리를 합니다.

## 왜 만들었는가

필름 카메라로 찍은 사진에는 디지털 사진에 없는 특유의 색감이 있습니다. 색 번짐, 그레인, 톤 커브 — 이런 요소들이 합쳐져서 "필름 느낌"을 만듭니다.

**이 프로젝트는 필름의 물리적 특성을 수학적으로 시뮬레이션하는 웹 기반 사진 에디터입니다.**

서버가 필요 없습니다. 브라우저만 있으면 됩니다. 이미지를 열고, 필름을 고르고, 보정하고, 내보내기까지 전부 클라이언트에서 처리합니다.

## 기술 스택

| 영역 | 기술 |
|------|------|
| UI | React 19 + TypeScript |
| 빌드 | Vite 7 |
| 스타일링 | Tailwind CSS v4 |
| 이미지 처리 | Canvas API (순수 픽셀 연산) |

백엔드 없이 단일 SPA로 동작합니다. 크로스 플랫폼을 위해 웹 기술을 선택했습니다.

## 주요 기능

### 필름 프리셋 선택
하단 필름 스트립에서 카테고리별로 필름을 탐색합니다. 썸네일 미리보기로 효과를 확인한 뒤 클릭하면 적용됩니다.

### Before / After 비교
상단 "Before / After" 버튼을 누르면 슬라이더가 나타납니다. 좌우로 드래그하면 원본과 편집 결과를 실시간 비교할 수 있습니다.

### 보정 슬라이더

**Light 그룹**: Exposure, Brightness, Contrast, Highlights, Shadows

**Color 그룹**: Saturation, Temperature, Tint

**Effects 그룹**: Grain, Vignette, Fade

### 이미지 탐색
- 마우스 휠: 줌 인/아웃
- 드래그: 팬 이동
- 더블 클릭: 줌 리셋

### 내보내기
JPG, PNG 포맷으로 내보낼 수 있습니다. 파일명에 적용된 필름 프리셋 이름이 자동으로 붙습니다.

## 실행 방법

```bash
cd film-editor
npm install
npm run dev
```

브라우저에서 `http://localhost:5173`을 열면 됩니다.

### 빌드

```bash
npm run build
```

`dist/` 폴더에 정적 파일이 생성됩니다. 아무 웹서버에 올리면 됩니다.

## 필름 시뮬레이션 엔진 구조

필름 시뮬레이션은 픽셀 단위로 여러 단계의 변환을 순차적으로 적용합니다.

**처리 파이프라인 순서:**

1. **Exposure / Brightness** — 노출 보정 (2의 거듭제곱 스케일링)
2. **Contrast** — 128 기준 대비 조정
3. **Color Matrix** — 3x3 매트릭스로 채널 간 색상 변환
4. **Tone Curve** — R/G/B 채널별 5포인트 S-커브 적용
5. **Saturation** — 루미넌스 기준 채도 조정
6. **Temperature / Tint** — 색온도와 틴트 시프트
7. **Fade** — 섀도우 리프팅 (필름 바랜 느낌)
8. **Vignette** — 중심부 거리 기반 주변부 어둡게
9. **Grain** — 노이즈 레이어 오버레이

각 필름 프리셋은 이 파이프라인의 파라미터 조합으로 정의됩니다. 정리하면, **필름 시뮬레이션 = 색상 매트릭스 + 톤 커브 + 그레인 + 비네팅의 조합**입니다.
